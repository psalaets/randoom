# randoom

Apocalyptic random number generator

## Usage

```js
var randoom = require('randoom');

console.log(randoom());
```

### Other Use Cases

O(1) data compression algorithm

```js
var randoom = require('randoom');

var compressed = randoom(internet);
```

*May be lossy for some inputs*

## Install

`npm install randoom -S`
